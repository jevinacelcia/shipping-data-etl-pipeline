# Shipping Data ETL Pipeline

## Overview

This project implements an ETL (Extract, Transform, Load) pipeline in Python that consolidates shipping data from multiple CSV files into a unified SQLite database.

## Features

- Reads shipping data from multiple CSV files
- Cleans and transforms inconsistent datasets
- Merges shipment information across files
- Inserts products into a normalized SQLite database
- Prevents duplicate product entries
- Verifies successful database population
